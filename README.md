# Reciplease

Indicate the ingredients you have and Reciplease will find all kinds of recipes you could make with them.  
Are you lacking inspiration? Ask Reciplease!

## Requirements
- iPhone with iOS 13+
- Data network 

## Foundations
- MVC
- Programmatic UI
- CocoaPods

## Frameworks
- UIKit
- Alamofire
- Core Data
- XCTest

## UX improvements
- Dark Mode
- Accessibility (Dynamic Type)
